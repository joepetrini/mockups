About These Mockups

These mockups demonstrate how the quote management functionality from the reference application will be implemented within the Loome ERP system. Each mockup includes:

  • Red Annotations - Explanations of key features and business logic
  • Interactive Elements - Working demos of autocomplete, calculations, and navigation
  • Tabler.io Styling - Consistent with Loome's existing UI framework
  • Business Rules - Visual indicators showing requirements and validations

Key Features Demonstrated

Business Logic
  • Automatic version ID generation (Q-yymm-xxxxxx)
  • Real-time price calculations with multipliers
  • Tax integration with ZIP code lookup
  • Freight calculation (% or $ toggle)
  • Margin and profit tracking
  • Credit limit validation
User Experience
  • HTMX-powered dynamic updates
  • Product autocomplete with keyboard navigation
  • Collapsible sections for better organization
  • Side-by-side version comparison
  • Template system for quick starts
  • Internal vs customer PDF options
